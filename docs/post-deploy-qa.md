# Post Deploy QA

デプロイ後に、以下を上から順に確認してください。

- [ ] デプロイ済みURLを開き、画面が正常表示される
- [ ] 楽天API未接続の場合、画面上部ヘッダーに「デモ表示中 — サンプル/モックデータ」バッジが常時表示される
- [ ] データソースを `サンプルデータ` にして `PS5` を検索し、カードが表示される
- [ ] データソースを `楽天市場` にして `SONY` を検索し、カードが表示される（キー未設定時はモックカードが表示され、検索結果の上に理由が表示される）
- [ ] `楽天市場` で 0 件になるキーワード（例: `zzzz-no-result`）を検索し、0件案内が表示される
- [ ] 任意カードを比較に追加し、「この価格を仕入れに使う」で利益計算へクイック入力できる
- [ ] `比較中カードをCSV出力` が動作し、CSVを開いて文字化けしない
- [ ] 履歴を保存し、再読み込み後に「再開」でき、不要履歴を「削除」できる
- [ ] テーマ切替（Simple Pro / Soft Market / Dark Trader / Natural Board）が動作する
- [ ] 幅 375px（iPhone SE相当）で表示崩れがない
- [ ] カードのソースラベル（`公式API取得` / `検索表示から推定` / `検索リンク` / `手動追加`）が意図どおり表示される
- [ ] 画面内に「実API接続済み」などの誤解を招く表現がなく、楽天API未接続時は sample/mock ベースであることが確認できる

補足:
- APIキー未設定でも動作します（サンプル/モックデータ）。楽天市場の実データを使う場合のみ `SERVER_RAKUTEN_APP_ID` の設定が必要です（Cloudflare Pages のみ対応、`docs/setup-guide.md` 参照）。
- 価格は参考表示を含むため、最終判断は元ページ確認を前提にしてください。
