# ユーザーガイド（MVPデモ版）

## できること

- 商品名・型番・JAN・URLを入力して `まとめて探す`
- 画像つき価格カードを確認
- `比較に追加` で比較ボードへ集約
- 比較ボードの価格をワンタップで利益計算へ反映
- `元ページを見る` で出典を直接確認
- 手動でカード追加（URL/価格/送料/状態/メモ）
- 検索ショートカットで外部検索ページを即オープン
- テーマ切替（4種類）

## 基本フロー

1. 検索バーに商品情報を入力
2. `まとめて探す` をクリック
3. 気になるカードを `比較に追加`
4. 比較ボードの `この価格を仕入れに使う / 販売に使う` を必要に応じてクリック
5. 利益見込みパネルで最終確認
6. `元ページを見る` で実際の価格・送料・状態を確認

## ソースラベルの見方

- `公式API取得` : 公式API由来
- `検索表示から推定` : 検索表示由来の推定値
- `検索リンク` : 外部検索ページを開くためのリンク
- `手動追加` : ユーザー入力

## 注意点

- 利益見込みは推定値です。必ず元ページで最終確認してください。
- 検索表示由来の価格は断定値ではありません。
