# ユーザーガイド

## デモ表示について（最初に確認）

- 画面上部のヘッダーには「デモ表示中 — サンプル/モックデータ」バッジが表示されます（スクロールしても残ります）。楽天市場 公式APIから実データを取得できた場合のみ、このバッジは緑色の「公式データ取得中 — 楽天市場」に自動で切り替わります。
- 各カードには `サンプルデータ`（UI確認用の固定データ）または `モック（楽天API想定）`（楽天API接続を想定した擬似データ）のバッジが付きます。
- 検索バー下の「データソース」で `サンプルデータ` / `楽天市場` を切り替えられ、それぞれの意味が一行で表示されます。現在のデータソースは右側のステータスでも確認できます。
- 楽天APIキー（`SERVER_RAKUTEN_APP_ID`）をサーバー側に設定した場合のみ、`公式API取得` ラベルの実データに切り替わります（未設定・タイムアウト・通信失敗・レート超過・上流エラー時は自動でモックに戻り、検索結果の上に理由が表示されます）。
- 検索中はボタンが「検索中…」と表示されて無効化され、二重に検索を送信できません。

## できること

- 商品名・型番・JAN・URLを入力して `まとめて探す`
- 画像つき価格カードを確認
- `比較に追加` で比較ボードへ集約
- 比較ボードの価格をワンタップで利益計算へ反映（反映元のカードが仕入れ・販売価格欄の下に表示されます）
- `元ページを見る` で出典を直接確認
- 手動でカード追加（タイトル/URL/価格/通貨（JPY・USD）/送料/状態/メモ）
- 検索ショートカットで外部検索ページを即オープン
- テーマ切替（4種類）
- 比較カードのCSV出力（データ種別・検索状態・警告を含む利益設定サマリー付き）
- リサーチ履歴の保存 / 再開 / 削除（localStorage、最大20件）
- ルールベースAIメモ（APIコストなし）

## 基本フロー

1. 検索バーに商品情報を入力
2. `まとめて探す` をクリック
3. 気になるカードを `比較に追加`
4. 比較ボードの `この価格を仕入れに使う / 販売に使う` を必要に応じてクリック
5. 利益見込みパネルで最終確認
6. `元ページを見る` で実際の価格・送料・状態を確認
7. 必要に応じてCSV出力し、履歴保存して後で再開

比較ボードに追加したカードは、新しく検索しても消えません。複数回の検索結果を横断して比較できます。検索語を変えて別の候補を探しながら、気になったものだけを比較ボードに残していく使い方を想定しています。

## CSV出力の列

比較ボードのカードごとに、商品名・サイト名・データ種別（実データ／サンプルデータ／モック）・
ソース区分（公式API取得／検索表示から推定／検索リンク／手動追加）・価格・通貨・送料・状態・
信頼度・元URL・メモ・取得日時を出力します。末尾のサマリー欄には、生成日時・データソース・
検索状態・検索日時・警告・利益設定・利益見込みをまとめて記載します。
`=` `+` `-` `@` から始まるセルは、表計算ソフトが数式として実行しないよう自動的に無害化されます。

## ソースラベルの見方

- `公式API取得` : 公式API由来
- `検索表示から推定` : 検索表示由来の推定値
- `検索リンク` : 外部検索ページを開くためのリンク
- `手動追加` : ユーザー入力

## 画面・操作のポイント

- 見た目をグラス（半透明＋背面ブラー）デザインに刷新しました。情報（ソースラベル・推定表記・状態）は従来どおり表示されます。
- すべての操作要素はキーボードの Tab で操作でき、フォーカス位置がアクセント色の枠線で見えます。
- OS の「視差効果を減らす」「透明度を下げる」設定に対応し、アニメ停止・不透明面へ自動で切り替わります。
- 幅 375px（スマホ）では右サイドのパネルが縦積みになります。

## 利益見込みの前提

- 仕入れ価格・販売価格・送料は0以上、手数料率は0〜100%の範囲でのみ入力できます（範囲外や数値でない入力は自動的に補正されます）。
- 手数料は「率（%）」のみを計算します。取引ごとの固定手数料（定額費用）は含まれません。該当する場合は送料や仕入れ価格に加算してください。
- 消費税・関税等は含まれません。

## リサーチ履歴について

- 保存できる履歴は最大20件です。上限を超えると、最も古い履歴から自動的に削除されます。
- 保存時点のデータソース・検索状態も一緒に記録され、履歴一覧に表示されます（例:「楽天市場（モック（キー未設定））」）。「再開」してもその時点の検索結果を復元するだけで、実際に再検索するわけではありません。
- ブラウザの保存容量が上限に達すると保存に失敗し、画面にエラーメッセージが表示されます。その場合は不要な履歴を削除してから再度お試しください。

## 注意点

- 利益見込みは推定値です。必ず元ページで最終確認してください。
- 検索表示由来の価格は断定値ではありません。
- AIメモはルールベースの注意喚起です。判断補助として使ってください。
