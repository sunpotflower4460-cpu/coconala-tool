# ユーザーガイド

## デモ表示について（最初に確認）

- 画面上部のヘッダーには常時「デモ表示中 — サンプル/モックデータ」バッジが表示されます（スクロールしても残ります）。
- 各カードには `サンプルデータ`（UI確認用の固定データ）または `モック（楽天API想定）`（楽天API接続を想定した擬似データ）のバッジが付きます。
- 検索バー下の「データソース」で `サンプルデータ` / `楽天APIモック` を切り替えられ、それぞれの意味が一行で表示されます。現在のデータソースは右側のステータスでも確認できます。
- 楽天APIキー（`SERVER_RAKUTEN_APP_ID`）をサーバー側に設定した場合のみ、`公式API取得` ラベルの実データに切り替わります（未設定・失敗時は自動でモックに戻ります）。

## できること

- 商品名・型番・JAN・URLを入力して `まとめて探す`
- 画像つき価格カードを確認
- `比較に追加` で比較ボードへ集約
- 比較ボードの価格をワンタップで利益計算へ反映
- `元ページを見る` で出典を直接確認
- 手動でカード追加（URL/価格/送料/状態/メモ）
- 検索ショートカットで外部検索ページを即オープン
- テーマ切替（4種類）
- 比較カードのCSV出力（利益設定サマリー付き）
- リサーチ履歴の保存 / 再開 / 削除（localStorage）
- ルールベースAIメモ（APIコストなし）

## 基本フロー

1. 検索バーに商品情報を入力
2. `まとめて探す` をクリック
3. 気になるカードを `比較に追加`
4. 比較ボードの `この価格を仕入れに使う / 販売に使う` を必要に応じてクリック
5. 利益見込みパネルで最終確認
6. `元ページを見る` で実際の価格・送料・状態を確認
7. 必要に応じてCSV出力し、履歴保存して後で再開

## ソースラベルの見方

- `公式API取得` : 公式API由来
- `検索表示から推定` : 検索表示由来の推定値
- `検索リンク` : 外部検索ページを開くためのリンク
- `手動追加` : ユーザー入力

## 画面・操作のポイント

- 見た目をグラス（半透明＋背面ブラー）デザインに刷新しました。情報（ソースラベル・推定表記・状態）は従来どおり表示されます。
- すべての操作要素はキーボードの Tab で操作でき、フォーカス位置がアクセント色の枠線で見えます。
- OS の「視差効果を減らす」「透明度を下げる」設定に対応し、アニメ停止・不透明面へ自動で切り替わります。
- 幅 375px（スマホ）では右サイドのパネルが縦積みになります。

## 注意点

- 利益見込みは推定値です。必ず元ページで最終確認してください。
- 検索表示由来の価格は断定値ではありません。
- AIメモはルールベースの注意喚起です。判断補助として使ってください。
