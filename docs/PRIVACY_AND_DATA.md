# データの扱いについて

## 基本方針

本ツールは、購入者・エンドユーザーの個人情報や検索データを制作者側のサーバーへ
送信・収集する仕組みを持ちません。

## 保存されるデータ

- 検索結果・比較ボード・利益設定・リサーチ履歴は、すべて **ブラウザの localStorage**
  （お使いの端末・ブラウザ内）にのみ保存されます。
- 保存先は同一ブラウザ・同一端末に限られます。別の端末やブラウザからは参照できません。
- リサーチ履歴の保存件数は最大20件です。上限を超えると古い履歴から自動的に削除されます。
- ブラウザの「閲覧データを削除」等の操作を行うと、保存内容も削除されます。

## 外部への通信

- 楽天市場 商品検索APIを利用する場合のみ、検索キーワードが自分（購入者）のCloudflare
  Pages Functions 経由で楽天のサーバーへ送信されます。この通信は制作者を経由しません。
- 検索リンク機能（メルカリ・ヤフオク・eBay・Yahoo!ショッピング等）は、ユーザーが
  クリックした場合にのみ、それぞれの外部サイトへ遷移します。自動的な通信は行いません。
- アクセス解析・広告トラッキング等のスクリプトは組み込まれていません。

## APIキーについて

- 楽天Application ID等のAPIキーは、フロントエンド（ブラウザに配信されるコード）には
  一切含まれません。Cloudflare Pages のサーバー側環境変数としてのみ保存されます。
- ビルド成果物（`dist/`）にキー文字列が含まれないことは、`docs/qa-checklist.md` の
  自動チェック・手動チェックで確認しています。

## CSV出力

- CSV出力はブラウザ内で完結するクライアントサイド処理です。ファイルは購入者の端末に
  直接ダウンロードされ、制作者のサーバーを経由しません。

## 購入者ご自身の責任範囲

- 本ツールを購入者自身のサービスとして公開・運用する場合、そのサービスにおける
  個人情報保護法等の遵守は購入者の責任で行ってください（`TERMS.md` 第4条参照）。
- 表示される価格・在庫情報は各サイトの公開情報・APIレスポンスに基づくものであり、
  本ツールが独自に収集・保管するものではありません。
