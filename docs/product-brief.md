# Product Brief

## Name

Coconala Tool / Market Card Research App

## One-Line Definition

商品名・型番・JAN・URLから、eBay・Yahooショッピング・楽天・メルカリ・ヤフオクなどの商品候補を画像つき価格カードで並べ、元URL確認・比較追加・利益計算までできるリサーチ補助ツール。

## Target Users

- ココナラで物販系ツールを求める副業初心者
- せどり・輸出入・中古リサーチをする人
- eBay / メルカリ / ヤフオク / Yahooショッピング / 楽天などを横断して価格を見たい人
- 難しい設定より、見やすく早いリサーチ体験を求める人

## Core Promise

難しい相場調査を、画像つきカードで見える化する。

ユーザーが求めているのは、完璧なデータベースではなく、次の4つです。

1. 見える
2. 比べられる
3. 元ページに飛べる
4. 利益が分かる

## MVP Scope

- 商品名検索
- まとめて探すボタン
- ダミー価格カード表示
- 比較に追加
- 利益見込み表示
- 検索リンク生成
- 手動/URL追加
- デザインテーマ切替
- localStorage保存

## Out of Scope for MVP

- ログイン
- 課金
- 本格SaaS管理画面
- 高頻度・大量自動スクレイピング
- 価格の完全保証
- 法務・税務判断の代行

## Product Personality

- とっつきやすい
- 見やすい
- オシャレ
- 初心者でも怖くない
- でも実務に使える

## Key UI Phrase

`商品名を入れて、まとめて探す。画像つきカードで相場を見る。`
