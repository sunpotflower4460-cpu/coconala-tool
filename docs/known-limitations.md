# 既知の制限事項

- v0.1 Demo は、実マーケットプレイスのリアルタイムデータには未接続です。
- `サンプルデータ` は UI / 操作フロー確認用、`楽天APIモック` は将来の公式API接続シミュレーション用です。
- メルカリ・ヤフオクは検索リンク生成と手動確認 / 手動追加を中心に扱います。
- 検索表示由来の価格は `検索表示から推定` として扱い、確定価格ではありません。
- APIキーは v0.1 では不要ですが、将来の実API接続時はサーバー / Worker 側で安全に管理する前提です。
- 保存データは `localStorage` ベースのため、同じブラウザ内にのみ保持されます。
- CSVエクスポートはクライアントサイド処理であり、サーバー保存は行いません。
- 認証、課金、バックエンド永続化は v0.1 の対象外です。
