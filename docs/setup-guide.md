# セットアップガイド（ローカル実行）

## 前提

- Node.js 20 以上推奨
- npm が利用可能

## 手順

```bash
npm install
npm run dev
```

ブラウザで表示されたローカルURLを開くと利用できます。

## 品質チェック

```bash
npm run lint
npm run build
```

## 現在のMVPでサンプル/ダミーな部分

- `まとめて探す` 実行時のカードは現状サンプルデータ中心
- APIアダプターは安全な足場（stub/mock）のみで、実API通信は未実装

## なぜメルカリ・ヤフオクは検索リンク中心か

MVPでは安全性と規約配慮を優先し、メルカリ・ヤフオクは検索リンク生成と手動追加を中心にしています。  
高頻度自動スクレイピングは実装しません。
