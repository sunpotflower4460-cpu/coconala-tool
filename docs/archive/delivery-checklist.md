# 納品チェックリスト（ココナラ向け）

- [ ] `npm install` で依存関係が導入できる
- [ ] `npm run dev` で起動できる
- [ ] `npm run lint` が通る
- [ ] `npm run build` が通る
- [ ] 検索→カード表示→比較→利益計算の流れが動作する
- [ ] 手動追加後に比較・利益計算へ反映される
- [ ] ソースラベル（公式API取得 / 検索表示から推定 / 検索リンク / 手動追加）が表示される
- [ ] 検索リンクカードが外部ページ用と分かるUIになっている
- [ ] テーマ切替が動作する
- [ ] README / setup-guide / user-guide を同梱

## 提案できるサービス文（例）

> 商品名を入れるだけで、画像つき相場カードを見やすく比較。  
> 気になる出品を比較に追加し、利益見込みまでその場で確認できる、初心者向けリサーチ補助ツールを提供します。  
> 外部検索リンク・手動追加にも対応し、元ページ確認しやすい安心設計です。
