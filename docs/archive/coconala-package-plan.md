# Coconala Package Plan

## Product Positioning

このツールは、物販・せどり・輸出入リサーチをする人向けの「画像つき相場カード比較ツール」として販売する。

売り文句:

> 商品名を入れるだけで、価格候補を画像つきカードで見やすく比較。元ページ確認・手動追加・利益計算までできる、あなた専用のリサーチボードを作ります。

## Package Tiers

### Light

Target price idea: 5,000〜15,000円

Features:

- 商品名検索UI
- ダミー/手動価格カード
- 検索リンク生成
- 手動/URL追加
- 利益計算
- 1 theme
- localStorage保存

### Standard

Target price idea: 30,000〜80,000円

Features:

- Light features
- eBay API adapter
- Yahoo Shopping API adapter
- 3+ themes
- CSV export
- More polished setup docs

### Premium

Target price idea: 100,000円以上

Features:

- Standard features
- Search API cards
- AI comments
- Supabase保存 or light backend
- Custom category tuning
- Chrome extension / share flow candidate

## Delivery Requirements

Every delivery should include:

- README
- setup guide
- user guide
- data source limitations
- API key guide, if APIs are used
- delivery checklist
- sample search terms

## Sales Caution

Do not advertise as:

- 全サイト完全自動取得
- メルカリ完全自動スクレイピング
- 正確な最安値保証
- 法務・税務判断ツール

Advertise as:

- リサーチ補助ツール
- 画像つき相場比較ボード
- 公式API・検索リンク・手動追加を組み合わせた時短ツール
- 元ページで確認できる安心設計
