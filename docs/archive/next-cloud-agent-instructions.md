# 次フェーズ向け Cloud Agent 指示（推奨）

## 次に進めるべきフェーズ

1. 公式API実装（サーバー/Worker経由）
2. 検索APIカード実装（推定ラベル厳守）
3. CSVエクスポート
4. AIコメント（短文・非断定）

## API実装時の必須ルール

- フロントエンドにAPIキーを直書きしない
- 公式API呼び出しは Cloudflare Workers / Supabase Edge Functions / Next.js API Routes など経由
- レスポンスは `MarketCard` 型に正規化
- 推定価格は必ず `検索表示から推定` ラベルを表示

## メルカリ・ヤフオクの扱い（MVP継続方針）

- 検索リンク生成を基本にする
- 元ページ確認をユーザーに促す
- 必要に応じて手動追加で価格確定
- 高頻度自動スクレイピングは実装しない
