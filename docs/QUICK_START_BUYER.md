# クイックスタート（購入者向け）

このツールを受け取った方向けの、最初に読むガイドです。専門知識がなくても迷わないよう、
順番に進められる構成にしています。

## まず確認すること

- [ ] `README_FIRST.md`（このZIPの一番上のファイル）に目を通した
- [ ] 自分が購入したプラン（ライト / スタンダード / プレミアム）を把握している
- [ ] `TERMS.md`（利用規約）を確認した

## プラン別の進め方

### ライトプラン（ソースコードのみ）

デプロイ作業は含まれません。ご自身でホスティング先を用意する場合は、
`USER_GUIDE.md` と `docs/setup-guide.md`・`docs/deployment-guide.md`（`source/` フォルダ内）
を参照してください。Node.js の知識がない場合は、スタンダードプラン以上へのアップグレードもご検討ください。

### スタンダードプラン以上（デプロイ済み）

制作者から共有されたURLを開くだけで利用開始できます。以下の手順で動作を確認してください。

1. 共有されたURLをブラウザで開く
2. 商品名（例: 「PS5」）を入力して「まとめて探す」を押す
3. 気になる価格カードの「比較に追加」を押す
4. 右側の「利益見込み計算」で仕入れ・販売価格を確認する
5. 「元ページを見る」で実際の価格・在庫を確認してから判断する

楽天市場の実データ連携（スタンダードプラン以上）を依頼した場合、ヘッダーが緑色の
「公式データ取得中」と表示されていれば実データ接続済みです。表示されない場合は、
まだサンプル/モックデータで動作している状態なので、制作者にご確認ください。

## 困ったときは

- 操作方法がわからない → `USER_GUIDE.md` を参照
- 表示されるデータの意味がわからない → `USER_GUIDE.md` の「ソースラベルの見方」を参照
- 動作しない・エラーが出る → `SUPPORT_POLICY.md` のサポート範囲を確認のうえ、ココナラのメッセージ機能でご連絡ください
- 個人情報やデータの扱いが不安 → `PRIVACY_AND_DATA.md` を参照

## このツールでできないこと（誤解を避けるために）

- 複数サイトからの完全自動取得はできません（楽天市場のみ公式API対応、他は検索リンク・手動追加）
- 表示価格は確定価格ではありません。最終判断は必ず元ページで確認してください
- 消費税・関税・固定手数料は利益計算に含まれません
