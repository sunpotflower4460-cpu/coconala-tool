# ココナラ掲載文

## サービスタイトル案

相場カード比較ツールを作成します（CSV出力・履歴保存・利益見込み対応）

## サブタイトル案

商品名入力から比較・利益確認までを、初心者にもわかる画面で提供します。

## サービス説明文案

物販・せどり・越境販売向けに、価格候補をカード形式で見やすく比較できるツールを制作します。

APIキー不要で、sample/mock データで主要フローを確認できます。楽天市場 商品検索APIの実データ連携はスタンダードプラン以上で対応します。

- 商品名・型番・JAN・URLで検索
- 気になる価格カードを比較ボードに追加
- 利益見込み（仕入れ/販売/送料/手数料）をその場で確認
- 比較中カードをCSV出力
- リサーチ履歴をローカル保存して再開
- 手動追加を使って候補カードをすぐ補完可能
- 検索由来価格は「推定」として明示し、元ページ確認を促進

「使い方がわかりやすい画面」を重視し、納品後すぐにデモ・運用しやすい構成でお渡しします。

## 購入前の注意文案

- 本ツールは相場リサーチ補助用です。最終価格は必ず元ページで確認してください。
- 主要機能（手動追加 / CSV / 履歴 / 利益計算）は API キーなしで利用できます。
- 楽天市場 商品検索APIの実データ連携はスタンダードプラン以上（Cloudflare Pagesへのデプロイ）で対応します。
- メルカリ・ヤフオクは検索リンク + 手動確認を前提とした設計です。
- APIキーはフロントエンドに埋め込まず、必要時はサーバー側設定で安全に接続します。
- 高頻度スクレイピングは実装対象外です。
- 表示される価格は推定・参考値です。取引の判断は購入者の責任で行ってください。
