# v0.1 リリースノート

## 概要

v0.1 は Coconala デモ向けのリリース候補です。サンプルデータと楽天APIモックを使い、検索・比較・CSV出力・履歴保存の流れを安全に確認できます。

## 含まれるもの

- `v0.1 Demo` 表示と、実API未接続であることの明示
- `現在のデータ: サンプル / 楽天APIモック` 表示
- 楽天APIモック0件時の案内と候補キーワード提示
- デモモード説明パネル
- README / デプロイ / バイヤー引き渡し / 制限事項の v0.1 向け整理

## 含まれないもの

- 楽天市場などの実API呼び出し
- APIキー設定
- スクレイピング
- 認証、課金、クラウド保存

## デプロイQAの要点

- `npm run lint`
- `npm run build`
- デプロイ後に `PS5` と `SONY` の検索を確認
- 楽天APIモックで0件検索の案内表示を確認
- CSV出力と履歴保存 / 再開を確認
